Expense Tracker – Python Project

A command-line based expense management tool developed using Python.
The project helps users record, organize, and analyze their daily expenses efficiently while applying fundamental programming concepts.

1. Project Overview

This project allows users to:

Add expenses with amount, category, and date

Store all entries in a structured CSV file

View all expenses

Filter by category

Calculate total expenses
The tool is simple, fast, and built entirely using core Python.

2. Features

User-friendly CLI interface

Automatic CSV storage

Categorized expense tracking

Total expense calculation

Clean modular code without external libraries

3. Technologies Used

Python 3

CSV file handling

Functions & modular approach

Loops, conditionals, error handling

4. How to Run the Project

Install Python (if not already installed).

Open terminal inside the project folder.

Run:

python exp_tracker.py


Use the menu to add/view/manage expenses.
